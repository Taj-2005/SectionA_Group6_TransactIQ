# Raw data drop zone

Place your raw dataset file(s) here, e.g.:

- `dirty_financial_transactions.csv`
- `dataset.json`

## Selected dataset (raw source)

- **Kaggle**: [Dirty Financial Transactions Dataset](https://www.kaggle.com/datasets/alfarisbachmid/dirty-financial-transactions-dataset)
- **Rule**: store the downloaded file(s) here **as-is**. Do not edit/clean/rename the source file contents; do all work in `data/processed/`.

## Requirements

- ≥ 5000 rows
- ≥ 8 columns
- Messy real-world data (missingness, inconsistencies, duplicates, outliers)
