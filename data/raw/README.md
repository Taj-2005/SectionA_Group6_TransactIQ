# Raw data drop zone

Place your raw dataset file(s) here, e.g.:
- `dataset.csv`
- `dataset.json`

## Requirements
- ≥ 5000 rows
- ≥ 8 columns
- Messy real-world data (missingness, inconsistencies, duplicates, outliers)

## Notes
- This repository ignores real data files by default (see `.gitignore`) to avoid committing large or sensitive data.
- Keep only small placeholders (like this README and `.gitkeep`) tracked in Git.
