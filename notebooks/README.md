# `notebooks/`

Notebook workspace organized by pipeline stage.

- `01_ingest/`: ingestion and schema checks
- `02_clean/`: cleaning + normalization + QA checks
- `03_eda/`: exploration and hypothesis building
- `04_features/`: feature engineering for analytics + ML
- `05_ml/`: modeling experiments (drop-off prediction, clustering, etc.)
- `06_exports/`: Tableau-ready exports and aggregates
- `_scratch/`: temporary work (do not rely on for final outputs)
